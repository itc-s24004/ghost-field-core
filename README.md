# Ghost Field Core

`ゴーストフィールド`のコアライブラリ

## 概要
このライブラリは、カードゲーム`ゴーストフィールド`のゲームロジックを実装するためのコアライブラリです。  
カードの効果、デッキ管理、プレイヤーステータス管理、ゲームの進行管理など、ゲームの基本的な機能を提供します。
